# WDSyncer

基于WebDav的安卓封装库

## 碎碎念

为啥要写这个库呢？

* 尝试自己写一个库调用，学习一下这个流程，为以后做准备
* WebDav同步过程很繁琐
* 自己做的项目都会涉及到同步功能，不想重复写代码
